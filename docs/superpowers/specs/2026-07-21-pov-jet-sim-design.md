# POV 戦闘機フライトシミュレーター 設計

作成日: 2026-07-21

## 概要
Web ブラウザで動く、一人称視点(POV)の戦闘機フライトシミュレーター。
ワイヤーフレーム(ベクター/レトロ調)のビジュアルで、自由飛行に加えて敵機との空戦を行う。

## 技術構成
- Three.js(CDN + importmap)。単一 `index.html` + ES モジュール数本。
- ブラウザで `index.html` を開けば即起動。ビルド不要。
- 配置: `~/dev/pov-jet-sim/`

## ビジュアル
- 全オブジェクト `wireframe: true`。暗背景 + ネオンライン。
  - 自機/UI = シアン、敵機 = 赤、地形 = 薄グリーン、弾 = 黄。
- 地形: ワイヤーグリッドの起伏(ハイトフィールド)。遠方はフォグでフェード。
- 撃墜時はワイヤーの破片が飛散する簡易爆散エフェクト。

## 飛行モデル(アーケード寄り)
- スロットルで前進速度を制御。ピッチ/ロール/ヨーで姿勢変更。
- 機首方向へ進む簡易物理。軽い重力と速度依存の揚力で「飛んでる感」。
- 失速・墜落(地面接触)で軽いリスポーン。

## 視点(C キーで切替)
- コックピット内視点: 計器枠 + キャノピー越しの前方。
- 機首まわり HUD のみ: 枠なしの没入 + HUD。

## HUD
- 速度・高度・方位(ヘディング)・スロットル・中央照準。
- 簡易レーダー(自機中心の敵位置)。
- 画面外の敵を指すロック/方向マーカー。

## 空戦
- 敵機ワイヤーフレーム数機(初期 4〜5 機)。
- 簡易 AI: 自機へ接近 → 一定距離で旋回・回避を織り交ぜる。
- 機銃(投射体)で撃墜。命中判定は距離ベース。
- 自機にも耐久値。被弾で減少、0 でリスポーン。

## 操作(両対応)
- キーボード: 矢印/WASD = ピッチ・ロール、Q/E = ヨー、Shift/Ctrl = スロットル増減、Space = 発射、C = 視点切替。
- マウス: ポインタロックで機首エイム、クリックで発射。

## モジュール分割
- `main.js` — シーン初期化、メインループ、カメラ切替、状態管理。
- `flight.js` — 自機の物理・姿勢・入力反映。
- `terrain.js` — ワイヤー地形生成と高度取得。
- `enemies.js` — 敵機生成・AI・撃墜処理。
- `weapons.js` — 弾の生成・移動・当たり判定・爆散。
- `hud.js` — HUD/計器描画(Canvas 2D オーバーレイ)。
- `input.js` — キーボード/マウス入力の集約。

## 成功基準
- ブラウザで開くと即操作可能で、フレーム落ちなく飛べる。
- 視点切替が機能する。
- 敵機を機銃で撃墜でき、被弾でリスポーンする。
- HUD に速度・高度・方位・レーダーが表示される。

## スコープ外(YAGNI)
- ミサイル/ロックオン誘導、複数武器切替。
- サウンド、マルチプレイ、スコア保存、モバイル対応。
- リアルな空力・燃料・ダメージモデル。
